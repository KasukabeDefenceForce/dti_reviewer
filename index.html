DTI Reviewer Documentation¶

DTI Reviewer is an intelligent expert finding system that helps identify subject-matter experts based on research abstracts and publications. The system uses advanced natural language processing and machine learning techniques to match research queries with relevant experts from a comprehensive database of academic publications.

Key Features¶

  • Transform free-form research abstracts into ranked lists of subject-matter experts

  • Advanced similarity matching using TF-IDF vector space model

Architecture¶

The system consists of two main components:

  • Frontend: A modern React-based web application

  • Backend: A Flask-based API server with similarity matching engine
